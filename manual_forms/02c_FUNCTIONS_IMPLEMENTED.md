### Специфика / сложности реализации

Перечисли всё, что неочевидно из spec/plan и важно помнить при сопровождении.
Примеры тем:

- **Кодировка ответа:** UTF-8 BOM → читаем через `encoding="utf-8-sig"`
- **Разделитель CSV:** `;` (не `,`) — указывать `sep=";"` в `pd.read_csv`
- **Лимит диапазона дат:** API режет запросы > 62 дней — делим на чанки
- **Rate limit:** submit не чаще 1/20 сек — между запросами `time.sleep(20)`
- **Пагинация:** курсорная по `nextPageToken`; пустой `rows` означает конец
- **Пустой ответ:** API возвращает `204` вместо `200` — обрабатываем отдельно
- **Группировка:** при `groupBy=NO_GROUP_BY` отсутствуют поля `date`/`campaignId`,
  возвращается одна строка-агрегат
- **Расхождения с доками:** поле `X` в реальности приходит как string, не number
